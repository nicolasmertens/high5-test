# Competitor Audit: Truity.com

**Date:** 2026-04-10
**Analyst:** Head of Growth Agent, 1Test
**Purpose:** Identify Truity's strengths, weaknesses, and exploitable gaps for 1Test

---

## 1. Homepage & Landing Page Analysis

### Value Proposition
- **Primary headline:** "Personality Tests & Career Assessments" (functional, keyword-focused)
- **Subheadline:** Free personality tests, career assessments, and relationship advice
- **Core promise:** Free tests with detailed results, career guidance, and relationship compatibility
- **Social proof:** 60M+ users claimed, media mentions (Forbes, BBC, Psychology Today)

### CTA Structure
- Multiple test cards on homepage — each framework gets its own card
- "Take the Test" CTA per framework — users must select which test to take
- No unified assessment option — 7 separate tests

### Conversion Funnel
1. Homepage → Choose a test (7 options)
2. Take 60-100 question quiz per test
3. Free basic results with personality type description
4. Premium upsell: detailed reports ($19-29 per test)
5. Ad-supported free pages with affiliate links and banner ads

## 2. Pricing Model

| Product | Price | Description |
|---------|-------|-------------|
| Free test (each) | $0 | Basic type description, no detailed analysis |
| Premium Enneagram Report | $19 | Full profile, wing analysis, growth paths |
| Premium DISC Report | $19 | Detailed behavioral profile, workplace insights |
| Premium 16 Personalities Report | $19 | Complete type description, career guide, relationship info |
| Premium Career Report | $29 | Career matches, workplace advice, industry-specific guidance |
| Premium Relationship Report | $19 | Compatibility with other types |
| Premium Team Report | $19/person | B2B team dynamics analysis |

### Key Insight
Each framework requires a separate payment. To get all four frameworks' premium results, a user pays $76 ($19 x 4). 1Test gives all four core results for free.

## 3. Content Strategy

### Content Volume
- 500+ blog posts and articles
- Individual type pages for all 16 MBTI types and all 9 Enneagram types
- Career guides per type
- Relationship compatibility pages
- "Type Explorer" interactive tool

### Key Content Properties
- Each type page (e.g., /intj, /enneagram-type-4) is a standalone SEO hub
- Long-form content (2,000-5,000 words per type page)
- FAQ sections on most pages
- Internal linking between type pages
- Regular blog content targeting personality-related keywords

### Keyword Strengths
- "personality test" — top 3 organic result
- "free personality test" — top 5 organic result
- Individual type keywords (e.g., "INTJ", "enneagram type 4")
- "DISC test", "enneagram test", "career test"
- Relationship compatibility terms ("INTJ compatibility", "Enneagram type 4 relationships")

### Content Weaknesses
- Blog content is informational-only, not conversion-focused until the CTA
- Free results are thin — designed to drive premium upgrades
- Ad-heavy free pages create poor UX (display ads throughout results)
- No cross-framework comparison content (can't compare your DISC to your Enneagram because they're separate products)

## 5. UX & Funnel

### Test Flow
- 7 separate tests, each 60-100 questions
- Slider-based answer format (11 steps per test page)
- No unified assessment — users must retake for each framework
- Results page shows basic type + prominent premium CTA
- Account creation optional (requires email for premium)

### Friction Points
- **333+ total questions** to complete all 4 frameworks separately
- **60-100 questions per test** — high abandonment risk
- **No cross-framework view** — can't see how your DISC relates to your Enneagram
- **Ad-heavy free results** — display ads throughout the free experience
- **Each premium report is a separate purchase** — $76 total for all 4

## Strategic Recommendations

1. **Lead with the one-test advantage**: "4 frameworks. 1 test. 120 questions. All free." This is the single most compelling differentiator. Truity requires 7 separate tests and 333+ questions. We do it in 15 minutes.

2. **Target Truity's weakness: fragmentation**: Create content around "DISC vs Enneagram", "personality test comparison", and "which test should I take" — terms Truity structurally cannot rank well for because they don't have cross-framework content.

3. **Target the free-results gap**: Every piece of messaging should contrast "free results for all 4 frameworks" against "pay $19-29 per framework". This is the most direct value proposition attack.

4. **Strengths is our unique vertical**: Truity does not offer a Strengths assessment. We own the "free Strengths test" keyword cluster outright. Invest heavily in Strengths content.

5. **Clean UX as differentiator**: No ads, no paywall popups, no "unlock your full report for $19" interruptions. The free experience should feel premium.

6. **SEO strategy: avoid Truity head-on, attack the flanks**: Don't compete for "free personality test" directly yet — Truity has years of domain authority. Compete for comparison terms, cross-framework terms, and Strengths-specific terms where they have no content.
